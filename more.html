How I Work

This is a glimpse into how I approach building projects and the tools I use. As an Information Technology Management student passionate about AI and sustainability, every step of my process is rooted in structure, creativity, and continuous learning.

🔧 My Workflow

🔍 Research
I define the problem, analyze its relevance, and gather data sources.
🧠 Brainstorm
I map out features and align with sustainability goals.
🧪 Prototype
Use Figma or sketches to visualize structure and flow.
🛠️ Build
I use Python, pandas, TensorFlow, and HTML/CSS for the core work.
📊 Analyze
Visualize and assess results using Tableau or model metrics.
🎯 Optimize
Refine using feedback, apply optimization (like SPEA2), and finalize.

🎯 Behind My Projects

CO₂ Forecasting & Sustainable Event Planning

This was my graduation project where I led a team to develop a smart platform for sustainable event planning. I used LSTM for CO₂ prediction, applied the SPEA2 algorithm to recommend venues, and designed the user flow to balance event cost and environmental impact.

OOP-Based Library Management System

An object-oriented Python system that simulates real-world book borrowing logic. I implemented class structures to handle borrowing, returning, and managing records with modular, clean code.
